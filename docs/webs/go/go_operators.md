# 操作符与运算符

运算符用于在程序运行时执行数学或逻辑运算。Go 内置的运算符分为：算术、关系、逻辑、位、赋值及其他运算符。

## 一、算术运算符

适用于数值类型（`+` 也适用于字符串拼接）。假定 `a = 10`、`b = 3`。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `+` | 相加 | `a + b` | 13 |
| `-` | 相减 | `a - b` | 7 |
| `*` | 相乘 | `a * b` | 30 |
| `/` | 相除 | `a / b` | 3（整数相除取整） |
| `%` | 求余 | `a % b` | 1 |
| `++` | 自增 | `a++` | 11 |
| `--` | 自减 | `a--` | 9 |

```go
package main

import "fmt"

func main() {
    a, b := 10, 3
    fmt.Println(a + b) // 13
    fmt.Println(a - b) // 7
    fmt.Println(a * b) // 30
    fmt.Println(a / b) // 3  整数除法向零取整
    fmt.Println(a % b) // 1

    a++
    fmt.Println(a) // 11

    // + 用于字符串拼接
    fmt.Println("Hello, " + "Go") // Hello, Go
}
```

**注意事项：**

- `++` / `--` 在 Go 中是**语句而非表达式**，只能单独成句、放在变量后面，不能写作 `b = a++` 或 `++a`
- 整数相除结果向零取整；除数为 0 会触发运行时 panic
- 浮点数除以 0 得到 `+Inf` / `-Inf` / `NaN`，不 panic
- Go **不支持幂运算符**，需用 `math.Pow`

```go
import "math"
fmt.Println(math.Pow(2, 10)) // 1024
```

## 二、关系（比较）运算符

返回布尔值。假定 `a = 10`、`b = 20`。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `==` | 是否相等 | `a == b` | false |
| `!=` | 是否不等 | `a != b` | true |
| `>` | 大于 | `a > b` | false |
| `<` | 小于 | `a < b` | true |
| `>=` | 大于等于 | `a >= b` | false |
| `<=` | 小于等于 | `a <= b` | true |

```go
a, b := 10, 20
fmt.Println(a == b) // false
fmt.Println(a < b)  // true

// 字符串也支持比较，按字典序（逐字符）比较
fmt.Println("abc" < "abd") // true
fmt.Println("Go" == "Go")  // true
```

## 三、逻辑运算符

用于布尔值。假定 `a = true`、`b = false`。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `&&` | 逻辑与，两边都为 true 才为 true | `a && b` | false |
| `\|\|` | 逻辑或，任一为 true 即为 true | `a \|\| b` | true |
| `!` | 逻辑非，取反 | `!a` | false |

```go
a, b := true, false
fmt.Println(a && b) // false
fmt.Println(a || b) // true
fmt.Println(!a)     // false
```

**短路求值：**

- `&&`：左侧为 `false` 时不再计算右侧
- `||`：左侧为 `true` 时不再计算右侧

```go
func check() bool {
    fmt.Println("check called")
    return true
}

func main() {
    if false && check() {
        // check() 不会被调用（短路）
    }
    if true || check() {
        // check() 不会被调用（短路）
    }
}
```

## 四、位运算符

对整数的二进制位进行操作。假定 `a = 60`（`0011 1100`）、`b = 13`（`0000 1101`）。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `&` | 按位与 | `a & b` | 12（`0000 1100`） |
| `\|` | 按位或 | `a \| b` | 61（`0011 1101`） |
| `^` | 按位异或 | `a ^ b` | 49（`0011 0001`） |
| `&^` | 位清除（AND NOT） | `a &^ b` | 48（`0011 0000`） |
| `<<` | 左移 | `a << 2` | 240 |
| `>>` | 右移 | `a >> 2` | 15 |

```go
a, b := 60, 13
fmt.Printf("%04b & %04b = %d\n", a, b, a&b)   // 12
fmt.Printf("%04b | %04b = %d\n", a, b, a|b)   // 61
fmt.Printf("%04b ^ %04b = %d\n", a, b, a^b)   // 49
fmt.Printf("%04b &^ %04b = %d\n", a, b, a&^b) // 48

fmt.Println(a << 2) // 240，左移 n 位相当于乘以 2^n
fmt.Println(a >> 2) // 15， 右移 n 位相当于除以 2^n
```

**要点：**

- `&^`（位清除）是 Go 特有运算符：`m &^ n` 等价于 `m & (^n)`，用于清除 `n` 中为 1 的对应位
- `^` 作为**一元运算符**表示按位取反：`^n`（等价其他语言的 `~n`）
- 左移 `<<` 补 0；右移 `>>` 对有符号数为算术移位（补符号位），对无符号数为逻辑移位（补 0）
- 移位位数必须为非负，运行时为负会 panic

```go
// 一元按位取反
var x uint8 = 0b0000_1111
fmt.Printf("%08b\n", ^x) // 11110000

// 有符号 vs 无符号右移
var s int8 = -8
var u uint8 = 248 // 与 -8 的补码位模式相同
fmt.Printf("%08b\n", uint8(s)>>1) // 逻辑移位补 0
fmt.Println(s >> 1)               // -4，算术移位保持符号
_ = u
```

## 五、赋值运算符

| 运算符 | 说明 | 等价于 |
| --- | --- | --- |
| `=` | 赋值 | — |
| `+=` | 相加后赋值 | `a = a + b` |
| `-=` | 相减后赋值 | `a = a - b` |
| `*=` | 相乘后赋值 | `a = a * b` |
| `/=` | 相除后赋值 | `a = a / b` |
| `%=` | 求余后赋值 | `a = a % b` |
| `<<=` | 左移后赋值 | `a = a << b` |
| `>>=` | 右移后赋值 | `a = a >> b` |
| `&=` | 按位与后赋值 | `a = a & b` |
| `\|=` | 按位或后赋值 | `a = a \| b` |
| `^=` | 按位异或后赋值 | `a = a ^ b` |

```go
a := 10
a += 5  // 15
a *= 2  // 30
a %= 7  // 2
a <<= 3 // 16
fmt.Println(a)
```

## 六、其他运算符

| 运算符 | 说明 | 示例 |
| --- | --- | --- |
| `&` | 取变量地址 | `p := &a` |
| `*` | 指针解引用 | `*p` |
| `<-` | 通道发送/接收 | `ch <- x` / `<-ch` |

```go
a := 42
p := &a          // 取地址
fmt.Println(*p)  // 42，解引用

ch := make(chan int, 1)
ch <- 100        // 发送
v := <-ch        // 接收
fmt.Println(v)   // 100
```

## 七、运算符优先级

Go 的二元运算符有 **5 个优先级**，由高到低如下；一元运算符（`!`、`-`、`^`、`&`、`*` 等）优先级最高。

| 优先级 | 运算符 |
| --- | --- |
| 5（最高） | `*` `/` `%` `<<` `>>` `&` `&^` |
| 4 | `+` `-` `\|` `^` |
| 3 | `==` `!=` `<` `<=` `>` `>=` |
| 2 | `&&` |
| 1（最低） | `\|\|` |

**要点：**

- 相同优先级的二元运算符**从左到右**结合，如 `x / y * z` 等价于 `(x / y) * z`
- 与其他语言明显不同：**移位 `<<` `>>` 优先级高于加减 `+` `-`**，且 `&`、`&^` 与乘除同级
- `++` / `--` 是语句不属于优先级体系，`*p++` 等价于 `(*p)++`
- 可用括号 `()` 提升子表达式优先级

```go
fmt.Println(2 + 3*4)      // 14，先乘后加
fmt.Println(1 << 2 + 1)   // 8：<< 优先级高于 +，即 (1<<2) + ... 不对！
// 注意：<< 高于 + ，所以 1 << 2 + 1 解析为 (1 << 2) + 1 = 5？
```

> 澄清上例：在 Go 中 `<<` 优先级（5）高于 `+`（4），故 `1 << 2 + 1` 实为 `(1 << 2) + 1 = 5`。这与 C 系语言（`<<` 低于 `+`）不同，是 Go 的重要差异点，建议对混合表达式**显式加括号**避免歧义。

```go
// 推荐显式加括号
result := (1 << 2) + 1 // 5，意图清晰
fmt.Println(result)
```
