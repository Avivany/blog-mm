# 操作符与运算符

运算符用于在程序运行时执行数学或逻辑运算。Go 内置的运算符分为：算术、关系、逻辑、位、赋值及其他运算符。

## 运算符语法

Go 的表达式语法由一元/二元运算符构成：

```ebnf
Expression = UnaryExpr | Expression binary_op Expression .
UnaryExpr  = PrimaryExpr | unary_op UnaryExpr .

binary_op = "||" | "&&" | rel_op | add_op | mul_op .
rel_op    = "==" | "!=" | "<" | "<=" | ">" | ">=" .
add_op    = "+" | "-" | "|" | "^" .
mul_op    = "*" | "/" | "%" | "<<" | ">>" | "&" | "&^" .

unary_op  = "+" | "-" | "!" | "^" | "*" | "&" | "<-" .
```

- 除位移运算外，二元运算符的两个操作数类型必须相同（无类型常量例外）
- 若其中一个操作数是无类型常量、另一个不是，则常量会被转换成另一个操作数的类型
- `++` / `--` 是**语句**而不是表达式，不在运算符优先级体系内

```go
x := 10
x++        // 合法：语句
*p++       // 等价于 (*p)++，因为 ++ 不在优先级体系
// y := x++ // 非法：++ 不是表达式，不能出现在赋值右侧
_ = x
p := &x
_ = p
```

## 一、算术运算符

适用于数值类型（`+` 也适用于字符串拼接）。假定 `a = 10`、`b = 3`。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `+` | 相加 | `a + b` | 13 |
| `-` | 相减 | `a - b` | 7 |
| `*` | 相乘 | `a * b` | 30 |
| `/` | 相除 | `a / b` | 3（整数相除取整） |
| `%` | 求余 | `a % b` | 1 |
| `++` | 自增 | `a++` | 11 |
| `--` | 自减 | `a--` | 9 |

```go
package main

import "fmt"

func main() {
    a, b := 10, 3
    fmt.Println(a + b) // 13
    fmt.Println(a - b) // 7
    fmt.Println(a * b) // 30
    fmt.Println(a / b) // 3  整数除法向零取整
    fmt.Println(a % b) // 1

    a++
    fmt.Println(a) // 11

    // + 用于字符串拼接
    fmt.Println("Hello, " + "Go") // Hello, Go
}
```

**整数除法与求余规则：**

对于整数 `x`、`y`，商 `q = x/y` 与余数 `r = x%y` 满足 `x = q*y + r` 且 `|r| < |y|`，**商向零截断**：

```
x     y     x / y     x % y
 5     3       1         2
-5     3      -1        -2
 5    -3      -1         2
-5    -3       1        -2
```

特例：当 `x` 为最小负数时，`q = x / -1` 仍等于 `x`（不会溢出变成正）。

| 类型 | 最小负数 |
| --- | --- |
| `int8` | `-128` |
| `int16` | `-32768` |
| `int32` | `-2147483648` |
| `int64` | `-9227483648`（实为 `-9223372036854775808`） |

**除零与负值位移：**

- 除数为常量时不能是 0；除数为运行时 0 会 **panic**
- 位移位数非负，运行时为负会 panic；除数（位移位数）为负数行为见下

**整数溢出（wrap around）：**

- 无符号运算 `+ - * <<` 是模 `2^n` 二进制运算，超出位宽会被丢弃（回绕）
- 有符号运算同样会溢出但并不 panic，结果仍以补码解释。因此 `x < x+1` 并非恒成立
- 浮点/复数运算 `+x == x`，`-x` 为其相反数

```go
// 整数除法向零截断
fmt.Println(-5 / 3) // -1
fmt.Println(-5 % 3) // -2

// 无符号溢出回绕
var u uint8 = 250
fmt.Println(u + 10) // 4（260 超出 uint8 位宽，回绕）

// 有符号溢出不 panic
var s int8 = 120
fmt.Println(s + 10) // -126（补码回绕）
```

**注意事项：**

- `++` / `--` 在 Go 中是**语句而非表达式**，只能单独成句、放在变量后面，不能写作 `b = a++` 或 `++a`
- 整数相除结果向零取整；除数为 0 会触发运行时 panic
- 浮点数除以 0 得到 `+Inf` / `-Inf` / `NaN`，不 panic
- Go **不支持幂运算符**，需用 `math.Pow`

```go
import "math"
fmt.Println(math.Pow(2, 10)) // 1024
```

## 二、关系（比较）运算符

返回布尔值。假定 `a = 10`、`b = 20`。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `==` | 是否相等 | `a == b` | false |
| `!=` | 是否不等 | `a != b` | true |
| `>` | 大于 | `a > b` | false |
| `<` | 小于 | `a < b` | true |
| `>=` | 大于等于 | `a >= b` | false |
| `<=` | 小于等于 | `a <= b` | true |

```go
a, b := 10, 20
fmt.Println(a == b) // false
fmt.Println(a < b)  // true

// 字符串也支持比较，按字节序（逐字节）比较
fmt.Println("abc" < "abd") // true
fmt.Println("Go" == "Go")  // true
```

**可比较性规则（规范要求）：**

- 比较双方类型必须可赋值给对方
- `==` / `!=` 要求操作数**可比较**；`<` `<=` `>` `>=` 要求操作数**可排序**
- 布尔值：可比较、可排序，同为 `true`/`false` 即相等
- 整数：可比较、可排序
- 浮点数：可比较、可排序（遵循 IEEE-754）
- 复数：可比较（`==`/`!=`），实部和虚部都相等即相等，但**不可排序**
- 字符串：可比较、可排序（按字节序）
- 指针：可比较、可排序；指向同一变量或同为 `nil` 时相等（指向未分配变量结果未定义）
- 通道：可比较；同一 `make` 创建或同为 `nil` 时相等
- 接口：可比较；动态类型相同且值相等时相等，或同为 `nil` 时相等
- 结构体：当**所有字段**都可比较时才可比较，非空字段全相等即相等
- 数组：当**元素**可比较时才可比较，所有元素全相等即相等
- 切片、map、函数值：**不可比较**；但它们与 `nil` 的比较是允许的

```go
const c = 3 < 4 // c 是无类型布尔常量 true

type MyBool bool
var x, y int
var (
    b3        = x == y // b3 类型为 bool
    b4 bool   = x == y // b4 类型为 bool
    b5 MyBool = x == y // b5 类型为 MyBool
)
_ = c
_ = b3
_ = b4
_ = b5
```

## 三、逻辑运算符

用于布尔值。假定 `a = true`、`b = false`。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `&&` | 逻辑与，两边都为 true 才为 true | `a && b` | false |
| `\|\|` | 逻辑或，任一为 true 即为 true | `a \|\| b` | true |
| `!` | 逻辑非，取反 | `!a` | false |

```go
a, b := true, false
fmt.Println(a && b) // false
fmt.Println(a || b) // true
fmt.Println(!a)     // false
```

**短路求值：**

- `&&`：左侧为 `false` 时不再计算右侧
- `||`：左侧为 `true` 时不再计算右侧

```go
func check() bool {
    fmt.Println("check called")
    return true
}

func main() {
    if false && check() {
        // check() 不会被调用（短路）
    }
    if true || check() {
        // check() 不会被调用（短路）
    }
}
```

## 四、位运算符

对整数的二进制位进行操作。假定 `a = 60`（`0011 1100`）、`b = 13`（`0000 1101`）。

| 运算符 | 说明 | 示例 | 结果 |
| --- | --- | --- | --- |
| `&` | 按位与 | `a & b` | 12（`0000 1100`） |
| `\|` | 按位或 | `a \| b` | 61（`0011 1101`） |
| `^` | 按位异或 | `a ^ b` | 49（`0011 0001`） |
| `&^` | 位清除（AND NOT） | `a &^ b` | 48（`0011 0000`） |
| `<<` | 左移 | `a << 2` | 240 |
| `>>` | 右移 | `a >> 2` | 15 |

```go
a, b := 60, 13
fmt.Printf("%04b & %04b = %d\n", a, b, a&b)   // 12
fmt.Printf("%04b | %04b = %d\n", a, b, a|b)   // 61
fmt.Printf("%04b ^ %04b = %d\n", a, b, a^b)   // 49
fmt.Printf("%04b &^ %04b = %d\n", a, b, a&^b) // 48

fmt.Println(a << 2) // 240，左移 n 位相当于乘以 2^n
fmt.Println(a >> 2) // 15， 右移 n 位相当于除以 2^n
```

**位移语义（重点）：**

- 移位运算符将左操作数移动「右操作数指定的位数」。右操作数必须是非负整数；运行时为负会 panic
- 左操作数有符号时为**算术移位**（补符号位），无符号时为**逻辑移位**（补 0）
- 移位**没有上限**，`x<<1 == x*2`、`x>>1 == x/2`（整数）
- 除位移外，若一方为无类型常量、另一方不是，常量转换为另一方的类型
- 右移表达式的右操作数必须是无符号整数或可转换为 `uint` 的无类型常量

```go
var s uint = 33
var i = 1 << s         // 1 类型为 int
var j int32 = 1 << s   // 1 类型为 int32；32 位平台下 j == 0
var k = uint64(1 << s) // 1 类型为 uint64；k == 1<<33
var n = 1 << s == 2<<s // 1 和 2 都是 int 类型；结果取决于 int 宽度
_ = i
_ = j
_ = k
_ = n
```

**一元运算符 `+ - ^` 定义：**

- `+x` 等价于 `0 + x`
- `-x` 取负，等价于 `0 - x`
- `^x` 按位取反，等价于 `m ^ x`（`m` 为全 1：无符号时 `m = ^T(0)`，有符号时 `m = -1`）

**要点：**

- `&^`（位清除）是 Go 特有运算符：`m &^ n` 等价于 `m & (^n)`，用于清除 `n` 中为 1 的对应位
- `^` 作为**一元运算符**表示按位取反：`^n`（等价其他语言的 `~n`）
- `x<<1` 等价于 `x*2`，`x>>1` 等价于 `x/2`

```go
// 一元按位取反
var x uint8 = 0b0000_1111
fmt.Printf("%08b\n", ^x) // 11110000

// 有符号 vs 无符号右移
var sv int8 = -8
var u uint8 = 248 // 与 -8 的补码位模式相同
fmt.Printf("%08b\n", uint8(sv)>>1) // 逻辑移位补 0
fmt.Println(sv >> 1)               // -4，算术移位保持符号
_ = u
```

## 五、赋值运算符

| 运算符 | 说明 | 等价于 |
| --- | --- | --- |
| `=` | 赋值 | — |
| `+=` | 相加后赋值 | `a = a + b` |
| `-=` | 相减后赋值 | `a = a - b` |
| `*=` | 相乘后赋值 | `a = a * b` |
| `/=` | 相除后赋值 | `a = a / b` |
| `%=` | 求余后赋值 | `a = a % b` |
| `<<=` | 左移后赋值 | `a = a << b` |
| `>>=` | 右移后赋值 | `a = a >> b` |
| `&=` | 按位与后赋值 | `a = a & b` |
| `\|=` | 按位或后赋值 | `a = a \| b` |
| `^=` | 按位异或后赋值 | `a = a ^ b` |

```go
a := 10
a += 5  // 15
a *= 2  // 30
a %= 7  // 2
a <<= 3 // 16
fmt.Println(a)
```

## 六、地址运算符

以类型 `T` 的 `x` 为操作数，取址 `&x` 生成类型为 `*T`、指向 `x` 的指针。操作数必须**可寻址**：可以是变量、指针解引用、切片索引、可取址结构体字段、可取址数组索引，或复合字面值。

```go
type Point struct{ X, Y int }

a := 10
arr := [3]int{1, 2, 3}

p1 := &a            // 变量地址
p2 := &arr[2]       // 数组/切片索引
p3 := &Point{2, 3}  // 复合字面值
_ = p1
_ = p2
_ = p3
```

对 `*T` 类型的 `x`，解引用 `*x` 表示其指向的 `T`；若 `x` 为 `nil`，解引用会 **panic**。

```go
var x *int = nil
// *x   // 运行时 panic
// &*x  // 运行时 panic（x 为 nil）
```

## 七、接收运算符（通道）

对于通道 `ch`，接收 `<-ch` 返回通道接收到的元素值，类型为通道元素类型。表达式会**阻塞直到有值**：

- 从 `nil` 通道接收会**永久阻塞**
- 从已关闭的通道接收，在所有数据取完后返回元素类型的**零值**
- 可用 `v, ok := <-ch` 形式，`ok` 为 `false` 表示通道已关闭且无数据

```go
ch := make(chan int, 1)
ch <- 100

v1 := <-ch          // 接收
fmt.Println(v1)
<-ch                // 阻塞直到有值并丢弃

// 带 ok 的接收
var v, ok = <-ch
_ = v
_ = ok
```

## 八、字符串拼接

字符串可用 `+` 与 `+=` 拼接，拼接会生成**新的字符串**：

```go
s := "hi" + string('!') // 拼接
s += " and good bye"
fmt.Println(s)
```

## 九、运算符优先级

Go 的二元运算符有 **5 个优先级**，由高到低如下；一元运算符（`!`、`-`、`^`、`&`、`*` 等）优先级最高。

| 优先级 | 运算符 |
| --- | --- |
| 5（最高） | `*` `/` `%` `<<` `>>` `&` `&^` |
| 4 | `+` `-` `\|` `^` |
| 3 | `==` `!=` `<` `<=` `>` `>=` |
| 2 | `&&` |
| 1（最低） | `\|\|` |

**要点：**

- 相同优先级的二元运算符**从左到右**结合，如 `x / y * z` 等价于 `(x / y) * z`
- 与其他语言明显不同：**移位 `<<` `>>` 优先级高于加减 `+` `-`**，且 `&`、`&^` 与乘除同级
- `++` / `--` 是语句不属于优先级体系，`*p++` 等价于 `(*p)++`
- 可用括号 `()` 提升子表达式优先级

```go
fmt.Println(2 + 3*4)      // 14，先乘后加
fmt.Println(1 << 2 + 1)   // 8：<< 优先级高于 +，即 (1<<2) + ... 不对！
// 注意：<< 高于 + ，所以 1 << 2 + 1 解析为 (1 << 2) + 1 = 5？
```

> 澄清上例：在 Go 中 `<<` 优先级（5）高于 `+`（4），故 `1 << 2 + 1` 实为 `(1 << 2) + 1 = 5`。这与 C 系语言（`<<` 低于 `+`）不同，是 Go 的重要差异点，建议对混合表达式**显式加括号**避免歧义。

```go
// 推荐显式加括号
result := (1 << 2) + 1 // 5，意图清晰
fmt.Println(result)
```
