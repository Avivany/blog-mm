# 《进阶》
### 重塑引人注目的高阶形象,第1章

1. <span style='background: #e6dcf9;'>一个成功的形象，展示出的是自信尊严与能力，它不但能够突出你的仪表风度，并且向公众传达着你的价值。虽然你不能改变容貌，</span>但完全可以重塑形象；就算你天生不美丽，却没人能阻止你修炼魅力。
2. <span style='background: #e6dcf9;'>进阶，从改变开始，</span> 换言之，过往发生在我们身上的一切错误，造成了今天梦想的贻误；
3. 形象如名片，没有它，你的自我展示就会大打折扣。事实上，所有气场强大的大企业家、行业领袖及政治家等。其形象都是经过专门塑造的。
4. 塑造形象不能局限于外表，而应更加注重塑造内在美；形象并不单单是指穿衣、发型、化妆等它是一个综合概念，是个人外在魅力与内在魅力的整体体现；形象并不局限于外在的形象，更包括人的思想追求抱负、价值观、人生观等。从某种意义上说，塑造形象就是与社会进行沟通，并为社会所接受的一种方式。在西方流传这样一句名言“你可以先将自己打扮成那个样子，直到自己成为那个样子。”使自己看起来更像个成功者，这更有助于你打开成功之门，让你脱颖而出；
5. 与众不同的特性是一个人走向成功的基础，忽视或抹杀自己的特性，是几乎不能真正成功的；
6. 尽管事件间的美好的东西比比皆是，但不在烙上自己特性的那片土壤辛勤付出的人，终将一无所获；
7. 在竞争激烈的时代，不展示自己的特性或个性，不拿出点自己的绝活儿，连生存都困难，更别谈发展和成功；
8. 有事在无关紧要的地方，不妨妥协和让步一下，减少不必要的麻烦；而在决定成败、决定前途和命运的关键时刻，须像雄狮和苍鹰那样独立，坚持自己独特的个性，高扬自己的特质，决不为任何外在的压力屈服；
9. 在生意场上，风度举止更为重要；
10. 高品位的生活方式绝不是粗俗、浮躁之人所能做到的，它需要的不仅是一种修养，更多的是一种心灵和精神的锤炼；
11. 强者风范是自信的一种体现，是铁打不弯的精神气概，是一种力量美和沧桑美。强者风范的内涵是顽强勇猛,坚毅果断,直而不肆,光而不耀。鲁迅说过,真的勇士，敢于直面惨淡的人生，敢于正视淋漓的鲜血。只有敢于面对现实、不屈不挠的人,才能铸就奋斗人生；
12. 强者风范与儒者心理之间没有千里鸿沟，强者更多是能够战胜自己的软弱，展现强者的一面；
13. 性格的美如艺术的美，在于它少有棱角，线条始终保持柔和；
14. 外貌的美是低层次的美，过于注重外在胜于内在的行为是不可取的；
15. <span style='background: #e6dcf9;'>宝石上光之后尽管更亮、更美，但它首先必须是颗宝石；</span>
16. 让对手也能看到你的风度：在这个社会，竞争与斗争都是我们无法逃避的现实，对我们来说，成功和失败也是不言而喻的，如何保持理智和风度则显得尤为重要；
17. <span style='background: #e6dcf9;'>能爱自己敌人的人是站在主动地位的人，采取主动的人是“制人而不受制于人”。你采取主动，不仅迷惑了对方，使对方搞不清你对他的态度，也迷惑了第三者，你的主动使对方处于“接招”“应战”的被动态势；</span>
